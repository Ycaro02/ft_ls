	WARNING Les : " sont a garder partout ou elles sont présentes
	Particulierement autour de LINK et PATH dans l'utilisation des commandes

creation d'un alias TEMPORAIRE aria2c="path_du fichier"
	doskey aria2c="D:\dldemor\aria2-1.19.0-win-32bit-build1\aria2c.exe" $*


Alias permanent
	Crée fichier :  d:\bat\macros.doskey
	Contenue: aria2c="d:\dldemor\aria2-1.19.0-win-32bit-build1\aria2c.exe" $*

	reg add "HKCU\Software\Microsoft\Command Processor" /v Autorun /d "doskey /macrofile=\"d:\bat\macros.doskey\"" /f
	reg query "HKCU\Software\Microsoft\Command Processor" /v Autorun

Utilisation :
	Utilisation basique, dl dans le dossier courant : aria2c "LINK"
	Pour changer le dossier de destination : aria2c "LINK" -d="PATH"
